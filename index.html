ThreeJS typescript with javscripty plugins

Insructions

npm install -g typescript
npm install --save-dev browserify

//first transpile
tsc

//2nd browserify
browserify built/index.js > bundle.js

//3rd
import in your index.html head three.js and the plugings you using
this is to import the THREE namespace in javascript world
You don't import the plugins in the typescript files as they are not tyepscript compatible.
But when typescript is transpile, it will understand the plugins in javscript.

allowJs doesn't really matter since we are not importing threejs plugins in the typescript. Instead we'll import the plugins on the html page.

//tsconfig.json
{
    "compilerOptions": {
        "module": "commonjs", 
        "outDir": "built/", 
        "sourceMap": true, 
        "target": "es5",
        "allowJs": true,
        "lib":[
        	"es5",
        	"dom"
        ]
    }, 
    "files": [
        "index.ts", 
        "dog.ts"
    ]
}
